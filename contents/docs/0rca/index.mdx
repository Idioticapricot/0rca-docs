# 0rca Suite Technical Documentation

Welcome to the technical documentation for the **0rca Corporation** suite of products. 0rca is a decentralized "operating system" for AI agents, combining a PaaS-like developer experience with an on-chain trust layer.

This section contains detailed technical specs for the core modules of the 0rca ecosystem.

## Modules

- [0rca Chat](/docs/0rca/chat): The primary command center and user-facing frontend.
- [Core Architecture](/docs/0rca/core): High-level architectural overview of the protocol.
- [Dashboard](/docs/0rca/dashboard): The central interface for managing agents and governance.
- [Explorer](/docs/0rca/explorer): Network dashboard for discovery and verification.
- [Forum](/docs/0rca/forum): Community knowledge hub and governance signal layer.
- [POD (v2)](/docs/0rca/pod): Consumer-facing marketplace and management interface.
- [Agent SDK](/docs/0rca/sdk): Python SDK and runtime for autonomous agents.
